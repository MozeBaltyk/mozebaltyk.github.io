## What is a workstation ?

A bundle with:
- OS 
- A user space
- Extras packages from OS repositories
- Extras packages from some other repos
- Some dotfiles

## The possibilites 

WSL + scripts

WSL + Cloud-init

Dockerfile to build a toolbox images

NixOS with flakes and a declarative config
