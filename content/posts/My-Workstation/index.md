## What is a workstation ?

## To summarized

A big bundle with:
- OS
- A user space
- Some extra-commands coming from OS repositories
- Some extra-commands from some other repos.
- Some .dotfiles
 
## How to reach this

WSL + shell scripts

WSL + Cloud-init

A Dockerfile to build A container toolbox 

NixOS with flake.nix and a declarative config file
